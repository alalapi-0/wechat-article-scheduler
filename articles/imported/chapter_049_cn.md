# 049 黄金の経験値【なろう版】 / 049 第４９話 「エキシビション」

2020/4/1

标题修订。

那个叫韦恩的玩家似乎也参加了活动。

突然被叫到名字，蕾亚吃了一惊。本来回应一下也无妨，但蕾亚和凯莉的声音完全不同，要是因此被胡乱猜疑就麻烦了。

于是她尽量压低声音念出启动关键词，发动魔法，将周围的玩家一并横扫。

蕾亚的『雷暴』造成的伤害极为严重，首先最前线的坦克角色全数阵亡。而稍后方，虽然装备了盾牌和厚重铠甲，但多为皮制、未穿戴金属胸甲的人则勉强存活。这大概是因为电击系魔法对金属防具具有特攻效果。

此外，或许是耐久力差距的缘故，包括韦恩在内的轻战士系玩家也死亡回到了复活点。他们原本就是以闪避为主的战斗风格，虽然AGI等属性较高，但将经验值分配到VIT上的人很少。

本来这类魔法应该是通过闪避来应对的，但电击系魔法从发动到弹速都快得惊人。虽然作为范围魔法，相比其他属性范围格外狭小，但也不是这种程度的玩家能躲开的。

剩下的，只有眼前的革甲坦克们，以及待在范围外的远程职业者。

蕾雅发动魔法后，战场上出现了一瞬间的空白，但很快，魔法与箭矢的暴雨恐怕又会倾泻而下。

虽然也可以朝远程职业者所在的方向再砸一发别的范围魔法，但在冷却时间结束前连续施放魔法，会让紧急关头能做出的选择越来越少。尤其是蕾雅刚才释放的那种高阶魔法，冷却时间特别长。

虽然麻烦，这里还是用剑来解决吧。

蕾雅一边将眼前的皮甲坦克一一斩杀，一边朝后卫方向冲去。负责移动的是铠坂同学，而挥剑的却是剑崎一郎本人。这也是在测试铠坂同学能否跟得上剑崎的动作。目前看来，并没有特别违和的地方。真是出色的配合。

将皮甲坦克一扫而空的铠坂同学，用左手拔出了背上的三郎。

她双手各持一把剑崎，提剑垂手，使出『缩地』逼近身旁的魔法师。趁对方惊慌失措、来不及做出任何反应之际，便砍飞了他的脑袋。看到那喷着血消失的尸体，周围的人才终于开始行动，但这里已经身处后卫集群的正中央。他们似乎都在犹豫该不该贸然射出箭矢或魔法，明显露出了迷惘。这想法真是毫无合理性可言。

虽说为了讨伐蕾雅，他们姑且统一步调，但这终究是场大逃杀。玩家之间原本就是彼此敌对的关系。既然如此，干脆抱着把所有人都一起炸飞的心态发动攻击不就好了。

如果能在那里打倒蕾亚那最好，就算打不倒，只要能用流弹击杀周围的玩家，应该也能拿到一些点数吧。这完全是毫无损失的选择。蕾亚实在无法理解，为什么没有人这么做。

与其说，换作蕾亚的话，比起去攻击那种如今再打一发魔法也感觉打不出伤害的敌人，她更会选择假装成流弹去攻击周围脆弱的玩家。那才是最合理的做法。

不过，如果被这么做，对现在的蕾亚来说可不是好事。因为如果其他玩家互相残杀，蕾亚能获得的经验值就会减少。

因此，在有人注意到之前，蕾亚会先解决掉自己周围的玩家。

她熟练运用【旋转斩】、【斩击】、【缩地】、【投掷】等技能，接连击杀玩家。被【投掷】出去的剑崎，会在不引起怀疑的范围内卷入数人并将其击杀，然后就那样插在地上。

投出剑的铠坂拔出背上的四郎和腰间的二郎，一边洒出普通攻击和技能，一边朝投出剑崎的方向前进，再将手里的剑崎【投掷】出去。接着用空出来的手回收刚才投出的剑崎。如此反复。

玩家们总算像是下定了决心，即使周围还有人，也开始朝这边施放攻击了，可到了这个时候，他们只剩下了零星几人。

「『闪耀箭』」

蕾亚厌恶抵消现象，对持弓玩家施展魔法，对施法玩家则进行投掷，逐一击杀。

这款游戏的攻击魔法设定有些特殊，当对来袭的魔法沿完全相反的轨迹施放魔法时，就会引发名为“抵消”的现象。如果是同属性且魔力量完全相同的攻击，通常会在碰撞点发生爆炸，无法抵达目标。此外，像是火属性与冰属性这类特定组合的魔法，威力会互相抵消，双方效果都消失殆尽。若因智力值差距等因素导致威力不同，或魔法等级有异，较弱一方会彻底消失，而较强一方在与较弱方的碰撞中损失的能量不会恢复，会以弱化后的状态继续飞向原本的目标。

像这样某人被集中针对、承受单方面攻击的混战，一旦演变为魔法战，很容易发生抵消和碰撞。

从当前状况来看，这单纯意味着蕾亚火力的下降。因此，既然要付出MP与冷却时间等成本和风险来施放魔法，就必须慎重斟酌。

不过，此时包围蕾亚的玩家们也已相当冷静。即使对弓系战士施放魔法，也常常有数名法师系玩家从背后放出魔法，将蕾亚的魔法完全抵消。

此外，也出现了能对蕾亚放出的“剑崎”射箭，将投掷武器的轨迹偏离这种高难度技巧的玩家。

这也让蕾亚感到惊讶。这类技能确实存在，但要对能够随意改变自身轨道的“剑崎”射箭命中且不让它回避，没有相当高超的现实操作技巧是做不到的。

蕾亚朝那边瞥了一眼，发现又是那个穿护士服的。她反射性地放出了魔法。

「『地狱烈焰』」

这一魔法将护士服所在的位置——从蕾亚视角看去的左手侧玩家们一扫而空。

范围魔法的威力完全无视些许抵消，其他玩家也因此停下了攻击。

尽管对蕾亚一人集中火力，战况却仍是平分秋色。

此刻一口气损失了近半数的战力，而残存者们又无法动弹，收拾起来便不费吹灰之力。

动摇的玩家联盟再也无法相互配合，大逃杀很快便落下了帷幕。

看来，表演赛的参加者们似乎无一例外都将目标对准了蕾亚。

就这样，第一届官方大型活动就此落下了帷幕。本来由运营策划的活动究竟是什么内容，最终还是无从得知。
