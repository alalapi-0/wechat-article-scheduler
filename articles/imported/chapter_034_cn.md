# 034 黄金の経験値【なろう版】 / 034 第３４話 「尻で椅子を磨くだけの簡単な」

2020/4/1

标题修订。

在让凯莉探查城镇期间，我让莱莉时不时地巡视大森林。

这座森林里应该已经没有蕾雅她们不认识的魔物了，但我还是让莱莉适当物色猫头鹰型的魔物，如果同种之中存在特别强大的个体，就让它毫发无伤地带回来。

在不需要查看凯莉视野的时候，我便共享莱莉的视野，观察森林的状况。

这样的蕾雅在「女王之间」里坐着的，自然是王座。

这是让工兵们像模像样地加工岩石制成后，再铺上好几张猎来的巨大魔物毛皮，布置到屁股不会痛的程度。与臀部接触的部分带有弧度，靠背稍微倾斜，增大了接触面积以分散体重，采用了类似旧世代汽车驾驶座的巧妙设计，即使长时间坐着也不会难受。

最近甚至会在这张椅子上睡着后直接登出。

紧邻旁边还有斯加尔专用的床，基本上斯加尔都在这里产卵。

产下后立刻进行教育，让它们学会物品栏，如果是指挥官级的话则学会好友聊天，然后编入部队。

一般来说，蚂蚁的巢穴变大后，生活范围也会随之扩大。因为只有从更广泛的范围收集食物，才能维持群体。

单就雷亚它们来说，由于没有幼虫和蛹的阶段，比普通蚂蚁更容易周转，但即便如此，当规模扩大到这种程度时，如果不把整个广阔的大森林都作为觅食地，就无法跟上需求。

掌握大森林也是出于这个原因。

通过将一部分魔物和野生动物等，以整个森林作为牧场进行饲养和管理，有些猎物反而比以往数量更多了。不过，那些魔物们大概根本没意识到自己被饲养着吧。

〈老大，我回来了〉

正当我通过劫持莱利的视觉和听觉，模拟享受探索森林的乐趣时，收到了凯莉发来的好友聊天。

恢复视觉和听觉后，发现凯莉已经回到了女王之间。她全身穿着廉价的兔皮铠甲。

「呵呵，你这身打扮是怎么回事啊。不，我不是说很奇怪啦。至少看起来像个像样的佣兵。」

「非常抱歉。我把资金用在了这种低级的装备上……」

「不过嘛，花的也就是凯莉靠伪装赚来的那点钱吧？就算花了也无所谓。我们本来就不怎么需要钱。至于装备，只要有一件上等的样品作为参考，现在我们也能照着它自己做出性能更高的装备。」

蕾亚她们甚至掌控着支撑这座城市周边金属供应的矿山。煤炭和木材等燃料与物资也是同样的情况。能获取上等毛皮的魔物，以及能产丝的芋虫类魔物，也基本只栖息在大森林中。

只要有样品作为参考，要做出比城镇品质更低的物品反而更困难。

不过，对于木材，倒也默许城里的樵夫砍伐森林外围的树木。

因此，森林外围绝对不会出现蕾亚的部下或牧场的魔物，应该被视为与魔物领域的边界线。

反正那片树木密度高的区域，似乎是因为低等级的树木不断生长、更新才形成这般模样。就算一次性向整座城镇供应木材，游戏内时间也只需短短十天左右就会恢复原样。

又或许，正是由于这种异常的生长周期，倒下的树木分解不及，才导致了地下泥炭层和煤层的形成吧。

「头儿，正如您刚才听到的，是关于那个所谓『大型活动』的事……」

「啊啊，是啊……。该怎么办呢。」

蕾雅也对凯莉与那个叫韦恩的玩家之间的对话中所提到的大型活动颇为在意。本以为以这款游戏的性质而言，此类活动会极为困难，但究竟要在何处、以何种方式举办呢。

首先得弄清楚活动的详细内容，否则连是否参加都无法决定。

再说了，不管怎样，要让蕾雅本人亲自前往某处参加活动，她都提不起兴致。

好不容易有这个机会，她倒是想照韦恩说的让凯莉去参加，但也不知道系统是否支持这种操作。

「总之，要是官方没有后续消息的话——」

〈老大，森林的情况好像有点不对劲〉

蕾亚话说到一半便停了下来。那是正在探索中的莱莉发来的好友聊天。

虽然“有点不对劲”这个报告确实很模糊，但莱莉应该不会无缘无故发出这样的报告。

〈不对劲？具体是怎么个不对劲法？〉

〈偶尔会看到被圈养的哥布林尸体。感觉数量比用来给蚂蚁新兵训练的多了一些。而且，似乎有我们不知道的气息……〉

莱莉本人似乎也没有什么确切的根据。报告的内容，如果说是误差，也还在误差允许的范围内。

看向旁边的斯加尔。由于已经指示过，关于支配区域的报告也要同时上报给斯加尔，因此陪同莱莉的蚂蚁应该也送来了同样的内容。

不过，斯加尔那边似乎也还没有收到什么特别的报告，于是摇了摇头。

这么说来，就是毫无预兆地突然出现了异物吗。

然而，在这片几乎可以说被完全掌控的森林里，在莱莉她们所在的靠近中心部的位置，实在难以想象会有谁能完全避开蚂蚁的警戒网而潜入进来。无论是陆地、天空，甚至地下，都有蚂蚁的存在。

〈老大！巡逻中的其他班传来报告！似乎突然出现了一支骨头军队！地点在W-18附近！〉

「骨头？是骷髅吗？突然出现是怎么回事。伤亡情况呢？」

〈我军目前没有伤亡。就在哥布林牧场附近，不过好像有几只前去警戒侦察的哥布林被干掉了。〉

搞不明白。W-18的话，从整片森林来看，是偏东侧、差不多正中央的位置。正如苏加尔所说，附近应该有哥布林牧场，但除此之外还有什么吗。

不，与其在这里思考，不如派人去侦察来得更快。正好莱利他们就在附近。

将视觉和听觉与莱利同步。

〈莱利，好像突然出现了一支骨头军队。不好意思，能请你率领小队去侦察一下吗？地点是W-18。〉

〈了解。〉

〈另外，如果那附近也有哥布林的尸体，那么敌人可能已经在那附近展开了。那些骨头家伙的隐蔽性不明，但务必多加注意。〉

〈是。〉

「斯加尔，哥布林牧场附近有没有什么可能与骨头军队有关的东西？」

〈好像有个哥布林们偶尔会去翻找武器之类的地方，但关于骨头……〉

「武器？　啊，确实它们拿着生锈的剑什么的……。感觉就像游戏里一样，氛围上没什么违和感，而且以哥布林的肌肉力量似乎也刺不穿蚂蚁的外壳，所以一直没在意，不过说起来，那些东西是从哪里弄来的？」

〈我们也没太在意，所以详细情况……。牧场旁边好像有个地方埋着大量快要朽烂的武器和铠甲……〉

要是让蚂蚁们在牧场附近太活跃，哥布林们会提高警惕，导致繁殖变慢，所以之前没靠近，但或许应该再多探索一下才对。

在建造延伸到牧场底下的地下通道时都没有发现，说明就算埋着，深度也不深，大概就在地表附近吧。

斯加尔没有上报，这意味着那些武器和铠甲即便被挖掘出来，等级也低于蕾亚她们现在能制作的装备品，不适合再利用。

只要牧场里的那些家畜用那些废品玩耍，我们大概就不会特别在意吧。

确实，就算真的上报了，只要没有发生像现在这样的异常事态，蕾亚大概也不会在意吧。

“算了。如果埋着大量腐朽的武装，那么原主人也埋在那里并不奇怪。虽然不知道是什么人，但源头已经有了线索。等莱莉的报告回来，确定之后我也出动吧。

如果是亡灵的话，大概我最能好好‘款待’它们了吧。”

而且这也是试试『死灵』新技能的好机会。
