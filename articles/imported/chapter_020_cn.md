# 020 黄金の経験値【なろう版】 / 020 第２０話 「教訓」

2020/4/1

标题已修订。

就在刚才，应该还能正常复活才对。不过因为那里蚂蚁成群，所以能完全顺利复活的例子很少。

「……可现在却因为是某人的私有地而无法复活，这难道是说在我看SNS的这几十分钟里，有人……买下了包括洞窟在内的土地吗？话说，土地能买吗？我要是作为骨头有了自信，也想买个房子什么的啊。」

但那个洞窟里应该有大量蚂蚁才对吧。即使有蚂蚁在，也能主张土地所有权吗？

如果真是这样的话，只要有资金，即使不战斗，怪物什么的也就不足为惧了。如果能动用国家预算，魔物的领域之类的应该能用征地开发一扫而空吧。

然而现实中，人类国家与魔物的领域正互相激烈地竞争着。

「正常想想，如果有原住民的话，土地所有权是不可能的吧。这样的话，是不是在我不在的这几十分钟里，有人把那个洞窟的蚂蚁全灭了？」

那种事情可能吗？就算有玩家在游戏服务开放的同时就开始玩了，到现在也才过了半天左右。在这么短的时间内，我不认为会有能把那些最强蚂蚁狩猎一空的玩家。

「啊，原来如此。说不定原本就生活在那世界里的人当中，就有那么强的家伙。好像说过玩家能做的事，NPC也大致都能做到。不如说NPC能做的事怪物应该也能做到，说不定本来就是蚂蚁自己制霸了那里。」

既然被称为蚂蚁，那存在类似蚁后的存在也不奇怪。

恐怕那个洞窟，是在蚁后的指示下，蚂蚁们正在进行压制的中途。布朗或许只是偶然出现在其中的异物而已。而且，恰好在布朗死亡期间，除了布朗以外的敌对势力全被一扫而空，于是那个洞窟彻底成了蚂蚁的乐园，布朗也就无法重生了。

「这还真‘蚁’有可能呢。毕竟是蚂蚁嘛。」

「但是，我死了而且处于接受复活中的状态，我的尸体应该在那里才对，不过如果蚂蚁制压之后有谁把我复活了，会发生什么呢……」

布兰想要确认帮助，但因为根本无法确定是关于哪方面的事情而烦恼。帮助中预想的问题太过多样，用模糊的搜索词是不可能到达目标帮助的。

「总之先查查复活时的条件，吧。」

苏生时的条件是：苏生对象的角色必须在死亡后一小时以内、苏生对象未被施加苏生限制等、尸体残留五成以上、并非处于苏生不可区域。

「可疑的应该是第三和第四个条件吧……」

尸体是否残留了五成以上不得而知。对布兰造成致命一击的攻击究竟是什么尚不清楚，但因为死法近乎即死，很难想象仅仅是被咬了一口。如果攻击手段是酸液，那么只要发动攻击的蚂蚁数量够多，短短几秒内就被溶解超过五成也不足为奇。

至于苏生不可区域，虽然并不清楚具体情况，但压制了洞窟的蚂蚁们有可能会做出这样的设定。毕竟布兰至少杀死了两只蚂蚁，对它们来说，不存在能无限重生的敌人显然是再好不过的。

不过这只是针对苏生的限制，与重生无关。

这样看来，果然应该认为布兰的尸体损失超过五成，在系统上从“布兰的尸体”转变成了“布兰的掉落物”。系统消息虽然写着“一小时内可接受苏生”，但如果真的找人帮忙苏生，大概会弹出错误之类的提示吧。苏生可以受理，但并不代表一定能复活。

总之，由于布兰留下掉落物后消失，周边一带满足了蚂蚁的制压条件，洞窟正式成了蚂蚁的乐园。

「大概就是这样吧……可恶，总有一天我要变强，把那座蚂蚁巢穴给攻略掉！」

不过，也不知道现在所在的洞窟离蚂蚁巢穴有多远。

在变强之前，必须先整顿好生活基础吧。可以说，正是因为缺乏这样的根基，才成了布兰输给蚂蚁的败因之一。

布兰目前的手牌只有魔法。想到对付蚂蚁时用『火魔法』都威力过剩了，如果对手是序盘敌人，或许即使不攻击弱点也能一击打倒。可是绝不能大意，不能忘记蚂蚁带来的教训。

“反正也习惯了，就做好心理准备，让自己随时都能立刻放出『烈焰箭』吧。那么，蚂蚁之后会是什么呢。”

布兰重新打起精神，一边警戒着洞窟一边开始前进。

洞窟比蚂蚁巢穴宽敞，也明亮许多。考虑到暗视能力一如既往地发挥着作用，原本大概只是有些昏暗吧。

“要是我也能把这里压制下来，是不是就能当成自己的家了呢？这里这么亮，要是能实现就好了。”

布兰将彻底征服这个洞窟定为了当前的目标。

只要没有其他玩家作为竞争对手，应该就绝非不可能实现。还有，如果敌人不是蚂蚁的话。

沿着洞窟前进，从某个地方开始，气氛突然改变了。

具体来说，从裸露的岩壁变成了似乎是用石头堆砌而成的人工墙壁。看起来，那道石墙非常古老，与岩石不同，大概是缝隙较多吧，到处都长满了苔藓。

“感觉像是遗迹……的氛围了啊。还好不是蚂蚁巢穴……不过遗迹里出现的野生动物会是什么呢？”

虽说只是序盘，但出现的敌人应该不限于野生动物，不过此时布朗因为初次见到的蚂蚁冲击太大，无意识中只想到了以现实生物为原型的魔物。

因此他惊慌失措，发出了叫声，来不及应对，就死了。完全没有成长。

“啊，有什么东西……。诶、哇！尸体！？是僵尸！？呀啊——！好恶心！这种针对性的伤害方式真的住手吧！”

《若在一小时内，可接受复活，是否立即重生？》
