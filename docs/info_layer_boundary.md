# 普通 / 详情 / 高级 三层信息边界清单

> Round 19 交付：为后续 Web 改造提供字段归属权威。  
> 实现轮次见各字段「落地轮次」列；未落地前以本清单为验收对照。

## 普通视图（首屏默认）

| 信息 | 普通用户说法（目标） | 当前状态 | 落地轮次 |
|---|---|---|---|
| 运行模式 | 演练 / 真实连接 | 仍显示 `mode: mock` | R20 / R25 |
| 是否会真发 | 不会真的发到公众号 | 英文开关文案 | R25 |
| 待办数量 | X 篇待发布 | 英文 `pending` 计数 | R20 / R21 |
| 主操作 | 1 找文章 → 2 排时间 → 3 执行 | 四个平铺按钮 | R22 |
| 最近结果 | 人话摘要 | 原始 key-value | R23 |
| 失败提示 | 原因 + 下一步 | 部分有 | R23 / R32 |
| 空状态引导 | 先把文章放进收件箱… | 技术腔 empty 文案 | R24 |

## 详情视图（点击文章/任务后）

| 信息 | 允许 | 禁止裸露 |
|---|---|---|
| 文章标题 | ✓ | 内部 id 作主标题 |
| 计划发布时间 | ✓ | — |
| 中文状态 | ✓ | 裸 `imported` / `failed` |
| 失败原因（人话） | ✓ | 原始 stack / adapter 码 |
| 是否有封面 | ✓ | 绝对路径作主文案 |

## 高级 / 调试视图（默认隐藏）

| 信息 | 说明 | 敏感 |
|---|---|---|
| 数据库路径 | 排错用 | 否 |
| 原始 JSON 响应 | scan/plan/run-once 完整 payload | 否 |
| 内部 id | article_id / job_id | 否 |
| `payload_json` | 事件审计原始体 | 否 |
| 英文枚举 | imported / pending / mock 等 | 否 |
| 内部统计 | processed / skipped_future / retry_count | 否 |
| secret / token / cookie | **禁止** | **是** |
