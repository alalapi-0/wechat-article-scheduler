# 封面资产设计

## 背景

真实微信 `draft/add` 依赖 `thumb_media_id`，需要稳定的封面素材来源。

## 当前状态

- 已实现：`RealWechatAdapter` 可上传默认封面（或 1x1 PNG 占位）
- 已实现：多目录素材扫描、stem 绑定、无效路径修复、孤儿清理（见 `docs/cover_asset_management.md`）
- 暂不做：完整封面裁剪与智能生成

## 规划

- 本地素材库（路径、尺寸、用途、来源）
- 文章与封面映射策略（默认、手选、回退）
- 素材审计事件（上传成功、替换、失效）

## 安全约束

- 不保存后台 cookie
- 不通过网页登录自动化管理素材
